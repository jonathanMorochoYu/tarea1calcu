MULTIPLICACION

Multiplicación: La multiplicación es una operación que combina dos o más números para obtener un producto. Se representa con el símbolo "×" o "*", y el resultado se llama "producto" La multiplicación se puede pensar como una forma abreviada de sumar varios números iguales entre sí. Por ejemplo, 3 × 4 es equivalente a sumar 3 cuatro veces: 3 + 3 + 3 + 3 = 12.
